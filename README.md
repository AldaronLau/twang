# Twang
Library for audio processing, conversions, and synthesis.

## Goals
- Auto-vectorized audio conversion and synthesis.
- Standard audio types
- Pure Rust

## Getting Started
Examples can be found in the [Documentation](https://docs.rs/twang) and the
examples folder.
